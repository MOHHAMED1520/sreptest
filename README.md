# عايد | STEP 2026 — أكاديمية عايد للغة الإنجليزية

## 🎯 وصف المشروع
موقع ويب تعليمي احترافي مخصص لتحضير اختبار كفايات اللغة الإنجليزية **STEP 2026** (النموذجان 51 و52)، مبني بهوية بصرية مستوحاة من شعار أكاديمية عايد (Navy #0F355C + Gold #F7AE1B).

---

## ✅ الميزات المنفذة

### 🎨 التصميم والهوية البصرية
- تصميم احترافي داكن مع هوية بصرية كاملة من شعار أكاديمية عايد
- شعار الأكاديمية ظاهر بارز في الصفحة الرئيسية داخل عرض دائري متحرك
- شريط أخبار متحرك (Ticker) ذهبي في أعلى كل صفحة
- شريط تنقل ثابت مع تأثيرات scroll
- Footer كامل مع روابط التواصل الاجتماعي
- متجاوب مع جميع الأجهزة (Responsive Design)
- أنيميشن سلسة: float, fadeUp, pulse-ring, shimmer

### 📄 الصفحات (4 صفحات)
1. **index.html** — الرئيسية مع Hero section + الشعار بارز + إحصائيات + خطوات + تقييمات
2. **summaries.html** — ملخصات تعليمية مفصّلة (Grammar, Reading, Vocabulary, Listening) بنظام Accordion
3. **quiz.html** — اختبار تفاعلي 100 سؤال مع تصحيح فوري
4. **courses.html** — صفحة دورات احترافية مع مقارنة وتقييمات

### 📚 بنك الأسئلة (100 سؤال تفاعلي)
- **Grammar:** 25 سؤالًا (أزمنة، مطابقة، شرطية، مجهول، مقالات، relative clauses، modals، gerunds)
- **Reading:** 25 سؤالًا (passages حقيقية + أسئلة main idea, detail, inference, vocabulary in context)
- **Vocabulary:** 25 سؤالًا (synonyms, antonyms, context clues, academic vocabulary)
- **Listening:** 25 سؤالًا (محادثات، محاضرات، إعلانات أكاديمية محاكاة)

### ⚡ نظام الاختبار
- تصحيح فوري بعد كل إجابة (أخضر = صحيح، أحمر = خاطئ)
- شرح تفصيلي لكل سؤال باللغتين
- شريط تقدم لكل قسم + إجمالي
- مؤقت لقياس وقت الاختبار
- Sidebar ثابت مع التنقل بين الأقسام

### 🏆 نظام النتائج
- نتيجة فورية بعد كل قسم مع تقييم مرئي (دائرة ملونة)
- توصية دورة مقنعة بعد كل قسم مباشرةً
- نتيجة كلية شاملة مع تفصيل الأقسام الأربعة
- توصية ذكية بالدورة المناسبة (تأسيسية/متوسطة/متقدمة) بناءً على النتيجة
- خطة مذاكرة 4 أسابيع مخصصة لكل مستوى
- أزرار مشاركة النتيجة (واتساب، تيليجرام، تويتر، نسخ رابط)

### 🎓 الملخصات التعليمية
- Grammar: أزمنة، مطابقة، شرطية، مجهول، مقالات، relative clauses، modals، gerunds
- Reading: Skimming & Scanning، Main Idea، Inference، أنواع الأسئلة
- Vocabulary: Context Clues، 20+ مفردة أكاديمية، مترادفات وأضداد
- Listening: الاستراتيجيات، Stress & Intonation، أنواع الأسئلة

### 📦 الدورات (courses.html)
- 4 دورات: تأسيسية (199 ر.س) + متوسطة (299 ر.س) + متقدمة (399 ر.س) + باقة شاملة (449 ر.س)
- جدول مقارنة تفصيلي
- فلتر حسب المستوى
- ضمانات واسترداد
- FAQ قابل للطي/نشر
- تقييمات الطلاب
